# Tracker Notifier

Esta aplicación tiene como intención rastrear envíos de ciertos proveedores y poder
notificar de cambios en el estado del envío.

## Características

* Rastreo de envíos
* Notificación vía Simplepush API
* Gestor web de ítems a rastrear
